# Example Directive Template

## Goal
Define what this directive accomplishes. Be specific about the expected outcome.

## Inputs
- **Input 1**: Description of what this input should contain
- **Input 2**: Format, type, or constraints on the input

## Tools/Scripts to Use
- `execution/script_name.py` - Brief description of what this script does
- Any other required tools or APIs

## Process
1. Step-by-step instructions for the agent to follow
2. Include decision points and conditional logic
3. Specify error handling approaches
4. Define validation requirements

## Outputs
- **Output 1**: Description of deliverable format and location
- **Output 2**: Where/how to store or present results

## Edge Cases
- **Case 1**: What to do when X happens
- **Case 2**: How to handle Y scenario
- **Rate Limits**: What rate limits exist and how to handle them
- **Error Scenarios**: Common errors and their solutions

## Success Criteria
Define what success looks like:
- Measurable outcome 1
- Measurable outcome 2

## Notes
Any additional context, limitations, or learnings discovered during execution.
